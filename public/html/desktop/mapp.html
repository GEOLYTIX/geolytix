MAPP

Our knowledge and data directly into your hands.

Secure and available whenever and wherever you are, MAPP allows spatial intelligence to be available for every decision.

  • Seamless

    Our consultancy services allow us to optimise the way you access spatial intelligence from our latest geodata packages and your own business metrics. Out in the field or at your desk the application is seamless. Key data attributes, locations, and imagery may be captured wherever internet access is available.

  • Customisable

    Build on top of an open source API, the MAPP interface provides customisable dashboards. These can be reports for export to PDF or functional widgets to be integrated in your website. Workspace configurations allow for the integration of models to be evaluated live and displayed within MAPP.

  • Secure

    Tiered security allows for the publication of open data as well as role based access. Applications can be accessible to the public with sensitive information being stored on different server accessible only via secure access. Roles can be assigned to registered user granting administrative rights to individual datasets or part thereof and the application itself.

  • Scalable

    Data and API hosted either on your premises or with a cloud services partner. Being able to deploy data and API as a service the application is scalable to run analytics processes synchronous. Maintained either by yourself or our expert systems engineers.
